XEL Documentation

What is XEL

XEL is an open source project creating a decentralized trustless super computer, utilizing cryptography and blockchain technology. The platform can execute a vast array of tasks in need of computational resources using the proprietary ElasticPL (ePL) programming language. Navigate through XEL’s tutorials and learn more on how you can submit a task , provide hardware or learn to program in ePL.
